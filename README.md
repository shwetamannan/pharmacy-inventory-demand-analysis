# Pharmacy Inventory & Demand Analysis

## Project Overview

This project analyzes multi-year pharmacy sales data across different medication categories to understand demand patterns, volatility, and inventory planning insights.

The goal is to help pharmacies improve stock management by identifying high-demand medications and understanding how demand changes over time.

---

## Dataset Description

The dataset contains pharmacy sales records across multiple therapeutic drug categories including:

- Pain relief medications
- Anti-inflammatory drugs
- Allergy medications
- Asthma/COPD medications
- Sleep and anxiety medications

Each row represents daily sales data.

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Key Analysis Performed

- Data cleaning and preprocessing
- Time series trend analysis (daily & monthly)
- Medication category comparison
- Demand volatility analysis
- Weekday vs weekend comparison

---

## Key Insights

- Analgesic medications showed the highest overall demand.
- Demand varies significantly across medication categories.
- No strong weekday-based pattern was observed.
- Some medications show high volatility, indicating unpredictable demand.

---

## Business Impact

- Helps optimize pharmacy inventory levels
- Identifies high-demand medications for better stocking
- Reduces risk of stock shortages
- Supports data-driven pharmacy operations
